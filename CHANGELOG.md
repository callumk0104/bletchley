# Changelog

All notable changes to Bletchley, newest first. Each release adds a short,
user-facing summary under a version heading; the release workflow pulls the
matching section into the GitHub release notes, and the app shows it under
"What's new".

## [Unreleased]
### Added
- Capture time to past or future weeks, not just the current one.
- Search past entries by description or timecode, with inline editing.
- "What's new" — view release notes from the menu and when an update is available.

### Changed
- The Timecodes view is now the home for Replicon sync, with a hide/show toggle
  to curate which codes appear in the picker.

## [1.0.2] - 2026-05-30
### Fixed
- In-app updates now install correctly when you click "Update to…".

## [1.0.1] - 2026-05-29
### Added
- Read-only Replicon timecode sync — pull your bookable Client / Project / Task list.

### Changed
- Timecodes now come from Replicon; removed the placeholder starter codes.

## [1.0.0] - 2026-05-28
### Added
- Initial release: Quick Capture, Weekly Grid with "Copy for Replicon", timecode
  manager, capture-now / resolve-later tray, daily target with live totals,
  end-of-day reminder, global hotkey + tray quick-add, automatic backups, and
  signed in-app auto-updates.
